## 摘要
经典鲁棒优化的不确定性集采用固定的几何形状，无法表示真实数据中存在的复杂依赖关系。我们提出生成式鲁棒优化（GRO），该框架通过深度生成模型将不确定性集定义为神经网络解码器在经校准的隐空间上的像，自然地容纳非线性相关性、非对称性和多模态性。一个五点评估框架（重构保真度、分布匹配、隐空间正则性、鲁棒相关性和计算可处理性）为评估任何基于神经网络的不确定性集提供了系统的、模型无关的标准。我们通过一个Wasserstein对抗自编码器实例化该框架，该编码器采用高斯混合模型引导训练以实现隐空间正则性，并通过约束一致性正则化实现鲁棒相关性。将解码器限制为ReLU激活函数使得通过混合整数规划嵌入进行精确最坏情况验证成为可能。针对一个生产规划问题进行的广泛实验涵盖了六种不确定性分布和六种生成架构，再加上一个多期设施选址研究，验证了该框架，并表明系统关注所有五个标准可产生同时具有表达力、良好校准性和优化可处理性的不确定性集。

## Abstract
Classical uncertainty sets for robust optimisation impose fixed geometric shapes that cannot represent the complex dependencies present in real-world data. We propose Generative Robust Optimisation (GRO), a framework in which a deep generative model defines the uncertainty set as the image of a neural network decoder over a calibrated latent set, naturally accommodating nonlinear correlations, asymmetry, and multimodality. A five-point evaluation framework (reconstruction fidelity, distribution matching, latent regularity, robust relevance, and computational tractability) provides systematic, model-agnostic criteria for assessing any neural network-based uncertainty set. We instantiate this framework with a Wasserstein Adversarial Autoencoder employing Gaussian mixture model-guided training for latent regularity and constraint-consistency regularisation for robust relevance. Restricting the decoder to ReLU activations enables exact worst-case verification through mixed-integer programming embedding. Extensive experiments on a production planning problem across six uncertainty distributions and six generative architectures, together with a multi-period facility location study, validate the framework and demonstrate that systematic attention to all five criteria yields uncertainty sets that are simultaneously expressive, well-calibrated, and optimisation-tractable.

### 1. 论文的核心问题与整体含义（研究动机和背景）
经典鲁棒优化（RO）采用固定几何形状的不确定性集（如盒子、椭球、多面体）来刻画不确定参数，但这些形状无法表示真实数据中常见的非线性相关性、非对称性、重尾和多模态结构。数据驱动方法（如统计假设检验构造集、核学习方法）虽有所改进，但仍然停留在**判别式**学习边界层面，且形状受限。分布鲁棒优化（DRO）则预定义模糊集形状（如Wasserstein球），同样缺乏形状自适应性。  
为此，本文提出**生成式鲁棒优化（GRO）**，利用深度生成模型学习不确定性参数的真实分布，并将其解码器的像定义为不确定性集，从而自然地适应复杂数据依赖，同时保持优化可处理性。

### 2. 论文提出的方法论
- **核心思想**：将不确定性集定义为训练好的解码器的像 \(U = \{G_\theta(z) : z \in Z\}\)，其中 \(Z\) 是隐空间中的简单有界集（盒或球），\(G_\theta\) 是经分布学习训练得到的解码器。通过校准隐空间边界（基于标准正态分位数），实现数据空间覆盖率的可控。
- **五点评估框架（P1–P5）**：  
  - P1：重构保真度（encoder–decoder路径能忠实还原样本）  
  - P2：数据空间分布匹配（生成样本分布接近真实分布）  
  - P3：隐空间正则性（编码后的聚合后验接近标准正态，保证校准有效性）  
  - P4：鲁棒相关性（解码器能够捕捉导致约束违反的尾部分布）  
  - P5：计算可处理性（解码器可精确嵌入MILP/MISOCP用于最坏情况验证）
- **实例化架构：WAAE-GMM-RO**  
  - **Wasserstein对抗自编码器**：结合WGAN-GP判别器、MMD损失、矩匹配损失，实现P1–P2。  
  - **GMM引导训练**：在数据空间拟合高斯混合模型，强制每个隐分量矩匹配标准正态（Corollary 1），提升P3。  
  - **约束一致性损失（P4）**：通过混合批次（训练数据+场景）并加权尾部分布，使用hinge/L1/MMD损失使解码器维持约束违反结构。  
  - **ReLU解码器 + 跳跃连接**：确保可精确编码为MILP（Big-M公式），满足P5。  
- **校准与求解**：  
  - 隐空间集采用盒约束 \(|z_j|\le\gamma\) 或球约束 \(\|z\|_2\le\gamma\)，半径由标准正态分位数确定（公式9-10）。

### 7. 优点
- **框架创新性**：首次整合深度生成模型与鲁棒优化，并系统化提出五点验证标准，具有方法学价值。  
- **可处理性**：通过ReLU解码器实现精确MILP嵌入，确保最坏情况验证的全局最优性，优于启发式方法。  
- **形状自适应**：解码器可适应非线性相关、多模态、偏斜，而传统形状无法捕捉。  
- **统一校准**：在标准正态隐空间上进行校准，覆盖度参数统一，无需针对每个问题推导解析形式。  
- **实验深度**：在多种分布与多场景下对比多个基线，包括分布鲁棒模型，验证充分。  
- **可重用性**：训练好的解码器可导出为ONNX块，跨问题复用（PP和FL共用架构）。

### 8. 不足与局限
- **计算成本高**：MILP/MISOCP求解时间随约束数量线性增长，FL实例中总墙钟达10.4小时（6线程并行），虽对战略规划可行，但对实时应用不友好。  
- **缺乏有限样本概率保证**：与Bertsimas等[2018]的检验方法相比，GRO没有提供给定样本量下的覆盖度保证，仅靠校准理论（Theorem 1）依赖隐空间接近正态的假设。  
- **隐空间校准依赖P3的完美性**：Theorem 1的KS界仅对边际成立；若隐空间聚合后验偏离正态，覆盖度偏差可能放大，实验中也观察到OBBT敏感性导致不同种子求解时间差异巨大。  
- **对高维不确定性的可扩展性未验证**：实验维度最高15（FL），但未讨论更高维（如100+）时解码器MILP规模爆炸问题。  
- **P4训练依赖下游问题**：需要生成决策池并预计算违反矩阵，增加了应用前的工作量。  
- **非线性约束未测试**：文中约束均为线性（预算、生产），对非线性约束（如黑箱函数）的MILP嵌入尚未验证。
